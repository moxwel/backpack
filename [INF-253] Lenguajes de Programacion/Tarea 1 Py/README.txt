Modo de uso:
El programa necesita un parametro para poder ejecutarse.
Es el nombre del archivo de las consultas que se van a realizar.

    Ejemplo:
        python ajax.py entradas.txt




Nota:
Carezco de habilidad para poder encontrar una forma de detectar
diccionarios dentro de diccionarios. Copie un RegEx dentro de si mismo.
Se pueden detectar un maximo de 20 diccionarios dentro de diccionarios.

Como extension, tambien se imprimira por pantalla los casos donde
las consutas no se puedan ejecutar porque el "data" que contienen esta mal
escrita especificamente. Esos casos son los equivalentes a decir "Consulta mal
escrita".
