==========
INFORMACION
==========

Esta pagina se desarolló en un entorno de Windows, utilizando las siguientes versiones
de estos programas:

- XAMPP 8.0.9 (https://sourceforge.net/projects/xampp/files/XAMPP%20Windows/8.0.9/)
- PHP 8.0.9 (http://localhost/dashboard/phpinfo.php)
- MariaDB 10.4.20 (http://localhost/phpmyadmin/)

Para la conexion a la base de datos:

- Ubicacion: localhost
- Usuario: root
- Contraseña: <nada>
- Base de datos: sansappdb

Si se desea modificar los datos de conexion (se tiene una contraseña o algo por el estilo), modifique
el archivo '/sansapp/php/database/db.php' con los datos correspondientes.

Entrego estos datos en caso de que ocurra un percance.



==========
EJECUCION
==========

1. Copie la carpeta 'sansapp' a la carpeta 'htdocs' de XAMPP.

2. Importe el esquema SQL que se encuentra en la carpeta 'bd' a phpMyAdmin.
   Una manera de hacerlo es arrastrar y soltar el archivo .sql dentro del navegador, con la
   pagina de phpMyAdmin abierta.

3. Ahora no debe haber problema al acceder a 'http://localhost/sansapp/'.
